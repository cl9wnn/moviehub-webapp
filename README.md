# MovieHub


A web application for discussion and communication between film enthusiasts.

## Features

- Users can create their own discussion topics, comment on others, and reply in threads.
- Explore films and actors, rate them, and add favorites to a personal wishlist.
- Email notifications are sent when someone responds to a user's comment.
- A content-based recommendation system generates a personalized movie feed based on user activity — including ratings, wishlist entries, preferred genres, created topics, and items marked as "not interesting."
- A personalized recommendations page for discussion topics suggests posts that match the user's interests and invites them to participate in active conversations.


## Technology Stack
- **Backend:** C#, ASP.NET Core 8, Entity Framework Core
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL
- **Object Storage:** MinIO 
- **Caching:** Redis
- **Background Jobs:** Hangfire
- **Containerization:** Docker
- **Testing:** xUnit, Moq
- **Logging & Monitoring:** Serilog + Seq
